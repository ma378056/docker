docker installing in linux :
-----------------------------------------------------
sudo yum update -y
sudo amazon-linux-extras install docker
sudo yum install docker
sudo service docker start

*.Add the ec2-user to the docker group so you can execute Docker commands without using sudo.
sudo usermod -a -G docker ec2-user    (after restarting the machine only it reflects)

*.Docker hub login
docker login -u "" -p ""

*.adding jenkins to docker group to avouid permission issue 
sudo usermod -aG docker Jenkins

*.docker login failed issue jenkin pipeline
sudo chmod 666 /var/run/docker.sock
